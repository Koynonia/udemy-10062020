## Criando uma conta para o Trello

| Passos | Descrição |
| :---: | :--- |
|1|Primeiro passo é acessar a página <https://trello.com> e criar uma nova conta.|
|2|Digite os seus dados no formulário e clique no botão para criar uma nova conta.|
|3|Após confirmar você vai ser redirecionado para a tela inicial do Trello, com um alerta em amarelo pedindo para você confirmar o seu endereço de e-mail, olhe na sua caixa de entrada do e-mail.|
|4|Confirme o seu e-mail clicando no botão conforme.|
|5|Para confirmar, você deve fazer login novamente, após essa etapa você vai ser redirecionado para a tela principal do Trello, sem o alerta em amarelo.|
|6|Para conhecer sobre as funcionalidades do Trello segue link do tutorial do próprio Trello: <https://trello.com/tour>|

## Criando uma conta para o Swaggerhub

| Passos | Descrição |
| :---: | :--- |
|1|Primeiro passo é acessar a página <https://app.swaggerhub.com> e criar uma nova conta.
|2|Você pode criar a sua nova conta usando uma conta do GitHub, para quem já tem, ou digitar os dados pedidos.
|3|Agora você deve definir o nome da sua organização, o nome da sua organização será parte da URL, clique no botão para avançar.
|4|Nesse passo você pode convidar algumas pessoas para fazerem parte da organização que você acabou de criar, você pode pular essa etapa, basta clicar no botão para avançar.|
|5|Na última etapa temos a página inicial do Swagger, temos as funções de criar uma nova especificação API ou importar.|
|6|Segue links da documentação da ferramenta: <https://app.swaggerhub.com/help/tutorials/getting-started#create-api>|

## Criando uma conta para o Apigee

| Passos | Descrição |
| :---: | :--- |
|1|Primeiro passo é acessar a página <https://apigee.com/api-management/#/homepage> e clicar para criar uma nova conta.|
|2|O segundo passo é preencher os dados do formulário e clicar no botão para submeter.|
|3|Acesse a caixa de entrada da sua conta de e-mail para confirmar o seu e-mail, o Apigee mostra um alerta após você clicar no botão de criar uma nova conta da etapa anterior.|
|4|Click no link que você recebeu no e-mail enviado.|
|5| Após clicar no link o Apigee mostrará uma página com o aviso de ativação da sua conta, isso pode levar alguns minutos, aguarde o processo finalizar, não feche o navegador apenas espere o processo.|
|6|Após finalizar o processo você vai ser redirecionado para <https://apigee.com/edge> se acaso ocorrer algum problema, clique no link anterior e faça o login para seguir para a tela inicial.|
|7|O link da documentação do Apigee é <https://docs-new.apigee.com/get-started>.|

## Criando uma conta para o GitHub

| Passos | Descrição |
| :---: | :--- |
|1|Primeiro passo é acessar a página <https://github.com/> e clicar na opção para criar uma nova conta.|
|2|Segundo passo é preencher os dados do formulário e clicar no botão para criar uma nova conta.|
|3|No segundo passo você deve clicar no botão **Create an account**.|
|4|Escolha a opção “Free” e clique no botão para continuar.|
|5|Na última etapa você pode preencher as opções pedidas e clicar no botão "Submit" ou clicar no link para pular essa etapa (*skip this step*).|
|6|Após a execução de todos os passos você terá acesso a página inicial do GitHub.|
|7|Link da documentação do GitHub é <https://guides.github.com/activities/hello-world/>|

## Criando uma conta para o Heroku

| Passos | Descrição |
| :---: | :--- |
|1|Primeiro passo é acessar a página <https://www.heroku.com/> e clicar na opção para criar uma nova conta.|
|2|Segundo passo é preencher os dados do formulário de cadastro e clicar no botão para criar a nova conta. Defina o Node.js como a sua linguagem primária.|
|3|Confirme a sua conta de e-mail clicando no link que foi enviado para o e-mail que você usou na criação da conta.|
|4|Após a confirmação do e-mail, você vai ser redirecionado para a página que você vai criar a sua senha.|
|5|Na página de confirmação, clique no botão para prosseguir.|
|6|Após clicar no botão do último passo você é redirecionado para a página principal do Heroku.|
|7|Link da documentação do Heroku é <https://devcenter.heroku.com/>|

## Criando uma conta no DockerHub

| Passos | Descrição |
| :---: | :--- |
|1|Acessar a página: <https://hub.docker.com/> e cliquem na opção para criar uma nova conta.|
|2|Preencham os campos e cliquem no botão **Sign Up**.|
|3|Vai ser pedido para você verificar a sua conta de e-mail para confirmar o e-mail.|
|4|Uma vez que você confirmou, usando o e-mail que você recebeu, você pode fazer Sign In no DockerHub.|
